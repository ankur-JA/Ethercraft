# Ethercraft 🛠️

A weekly learning and building journey into smart contract development using Foundry, Solidity, and Ethereum core concepts.

Each folder (`week_01`, `week_02`, ...) is a standalone Foundry project focused on one key topic, such as token standards, DeFi protocols, DAOs, or gas optimization.

## 📁 Structure

- `week_01/` → ERC20 Token + Vesting  
- `week_02/` → ERC721 NFT Minting Site  
- `week_03/` → DAO Governance with Governor  
- `week_04/` → Uniswap V2 Clone (DEX core logic)  
- ...and so on.

## 🧪 Stack

- [Foundry](https://book.getfoundry.sh/) for testing and deployment  
- Solidity for smart contract development  
- Optional: React + Wagmi/Viem/Ethers for frontend integrations

## 🚀 Why Ethercraft?

This repo documents my journey to becoming a smart contract developer. Each week is focused on building production-grade components, testing them thoroughly, and deploying to testnet.

### 🧱 Goals
- Write gas-optimized, secure smart contracts
- Learn ERC standards and DeFi patterns
- Build complete dApps with frontend integration
- Showcase progress to recruiters and hiring teams
